The following examples are from the HLS SEMINAR 2021.
The seminar material can be found in Early_Design_and_Validation_of_an_AI_Accelerator.pdf.

You will want to source the "set_vars" script file to download and build the variouse open-source components used in these examples.
  Example for C-Shell users:
    source ../set_vars.csh
  Example for Bourne Shell users:
    . ../set_vars.sh
